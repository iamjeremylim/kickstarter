# CrowdCoin

CrowdCoin is a kickstarter alike crowdfunding platform, built using Ethereum. With its smart contract functionality, campaign funds will only be deployed to vendors who are approved by the campaign contributors. This prevents misappropriation of campaign funds.

## App Features

- User can view a list of open campaigns
- User can create a campaign
- User can view a campaign
- User can contribute ether to a campaign
- Campaign owner can create request to send ether to vendors
- Campaign contributor can approve request to send ether to vendors
